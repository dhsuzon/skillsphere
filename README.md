# 🎓 SkillSphere – Online Learning Platform

## Project Overview
**SkillSphere** is a modern, intuitive online learning platform designed to help users explore, search, and manage their educational journey. Built with **Next.js 15**, it offers a seamless experience for students to find and enroll in high-quality courses across Web Development, Design, Marketing, and more.

## Project Live Link
  [https://skillsphere-eosin.vercel.app](https://skillsphere-eosin.vercel.app)

## 🛠 Tech Stack
- **Framework:** Next.js 15
- **Styling:** Tailwind CSS, DaisyUI, HeroUI
- **Animations:** Framer Motion
- **Authentication:** Better-Auth
- **Slider/Carousel:** Swiper.js
- **Icons:** React Icons, Lucide React

---

## ✨ Core Features
- **Secure Authentication:** Seamless login via Email/Password and Google Social Login powered by **Better-Auth**.
- **Protected Routing:** Premium content and course details are exclusively accessible to registered users.
- **Dynamic Search:** Real-time course filtering by title or category for a streamlined experience.
- **Personalized Profile:** Dedicated dashboard for users to manage their account details and profile pictures.
- **Interactive UI:** Smooth navigation and engagement using **Swiper.js** and **Framer Motion** animations.
- **Optimized Experience:** Responsive design for Mobile, Tablet, and Desktop; includes a custom 404 page for better UX.
- **Instant Feedback:** User actions are acknowledged via **React Hot Toast** notifications.
